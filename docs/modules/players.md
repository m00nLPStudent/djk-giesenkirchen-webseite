# Modul: Spieler

## Status

Abgeschlossen.

## Funktionen

- Spieler anlegen
- Spieler bearbeiten
- Spieler aktiv/inaktiv setzen
- Spieler löschen über zentrales Löschmodul
- Spielerbild verwalten
- Mannschaft zuordnen
- Spielerprofilseite
- Anzeige im Mannschaftskader
- Filter und Sortierung

## Datenbank

- `players`
- `player_team_seasons`

## Zielstruktur

Langfristig sollen doppelte Felder nicht in die Live-Datenbank übernommen werden. Siehe `docs/database-target-schema.md`.
