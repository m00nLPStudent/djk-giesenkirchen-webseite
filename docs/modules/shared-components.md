# Modul: Shared Components

## Ziel

Gemeinsame Komponenten vermeiden doppelte UI und erleichtern neue Module.

## Admin-Komponenten

- Karten
- Badges
- Formulare
- Tabs
- Uploads
- Löschdialoge
- Statistikboxen

## Website-Komponenten

- Navigation
- News-Karten
- Profilkarten
- Mannschaftssektionen

## Regel

Wenn eine Komponente in mehr als einem Modul gebraucht wird, soll sie in einen gemeinsamen Bereich verschoben werden.
