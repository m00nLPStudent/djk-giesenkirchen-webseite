# Datenbank Beziehungen

## departments zu teams

Eine Abteilung kann mehrere Mannschaften besitzen.

## teams zu team_seasons

Eine Mannschaft kann für mehrere Saisons eigene Daten besitzen.

## seasons zu team_seasons

Eine Saison kann mehrere Mannschaftseinträge besitzen.

## players zu player_team_seasons

Ein Spieler kann pro Saison einer oder mehreren Mannschaftskonstellationen zugeordnet werden.

## coaches zu coach_team_seasons

Ein Trainer oder Betreuer kann pro Saison einer Mannschaft zugeordnet werden.

## news zu teams

Eine News kann optional einer Fußballmannschaft zugeordnet werden.

## team_templates

Team-Vorlagen dienen nur zur Erstellung neuer Mannschaften und sind keine direkten Live-Inhalte.
