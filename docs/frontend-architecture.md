# Frontend Architektur

## Ziel

Die öffentliche Website soll modern, klar strukturiert und später vollständig responsive sein.

## Aktuelle Bereiche

- Startseite mit News
- News-Übersicht
- News-Detailseite
- Fußballbereich
- Mannschaftsdetailseiten
- Trainerprofile
- Navigation mit Dropdowns
- Footer mit Platzhaltern

## Gestaltungsprinzipien

- Vereinsfarben nutzen
- Dunkles modernes Layout
- Große visuelle Flächen
- Wiederverwendbare Karten und Sektionen
- Navigation klar und visuell hochwertig

## Responsive Strategie

Responsive Optimierung wird gesammelt und später geschlossen durchgeführt. Dadurch vermeiden wir doppelte Anpassungen, solange Module noch stark im Aufbau sind.

## fussball.de Widgets

Die Widgets sind technisch vorbereitet. Die finale Darstellung wird erst auf der echten Domain getestet und angepasst.
