# Deployment

## Ziel

Die Website soll später auf der finalen Domain stabil laufen und mit einer sauberen Produktionsdatenbank verbunden sein.

## Aktueller Stand

- Entwicklung lokal
- Quellcode in GitHub
- Supabase als Entwicklungsdatenbank
- fussball.de Widgets technisch vorbereitet

## Vor Livegang prüfen

- Produktionsdatenbank anlegen
- Umgebungsvariablen setzen
- Storage prüfen
- Domain konfigurieren
- Responsive Design prüfen
- SEO-Grundlagen prüfen
- Performance prüfen
- fussball.de Widgets auf echter Domain testen

## Regel

Der Livegang erfolgt erst, wenn Website, Adminbereich, Datenbank und Storage gemeinsam getestet wurden.
