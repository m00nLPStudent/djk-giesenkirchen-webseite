# Routes

Dokumentation der wichtigsten Routenstruktur.

## Website

- /news: aktuelle News in großer Darstellung
- /news/uebersicht: klassische Übersicht mit Suche und Pagination
- /news/[slug]: News-Detailseite inkl. Downloadbereich

## Admin

- /admin/news: News-Übersicht
- /admin/news/new: News anlegen
- /admin/news/edit/[id]: News inkl. Dokumente bearbeiten
